Important FAQ

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that allows scripts and webs to dynamically access and update the content structure, and style of a web document. It represents the HTML structure of a webpage as a tree of objects, which can be manipulated using JavaScript.

2. How do you select an element from the DOM?

You can select elements from the DOM using various methods provided by JavaScript, such as:

  • document.getElementById() selects an element by its ID.
  • document.querySelector() it will selects the first matched element based on a CSS selector.
  • document.querySelectorAll() it will selects all matching elements based on a CSS selector.
  • document.getElementsByClassName() it will selects elements by their class name.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element that listens for events from its children. Instead of adding an event listener to individual child elements, the event is "delegated" to the parent. This is useful because it reduces the number of event listeners.

4. How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles, you can use JavaScript methods like:

  • element.setAttribute() and element.getAttribute() to set and get attributes.
  • element.style.property to directly modify inline CSS styles. For example, element.style.color = 'tellow';
  • element.classList.add() and element.classList.remove() to add or remove CSS classes for style manipulation.
